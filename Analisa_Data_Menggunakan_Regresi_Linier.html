Analisa Data Menggunakan Regresi Linier#

1. Pendahuluan#

Regresi linear merupakan salah satu metode dalam statistika yang digunakan untuk melihat hubungan antara dua variabel, yaitu variabel bebas (X) dan variabel terikat (Y). Metode ini cukup sering digunakan dalam analisa data karena mampu membantu dalam proses prediksi maupun melihat pola hubungan antar data.

Pada regresi linear sederhana, hubungan antara variabel X dan Y digambarkan dalam bentuk garis lurus. Garis tersebut diperoleh dari hasil perhitungan sehingga dapat menunjukkan kecenderungan data yang dimiliki. Bentuk umum persamaan regresi linear sederhana yaitu:

\[ Y = \beta_0 + \beta_1X \]

Keterangan:

  • \(Y\) adalah variabel terikat

  • \(X\) adalah variabel bebas

  • \(\beta_0\) adalah konstanta (intercept)

  • \(\beta_1\) adalah koefisien regresi (slope)

Nilai intercept dan slope dapat dihitung menggunakan beberapa cara. Pada tugas ini digunakan dua metode, yaitu menggunakan library LinearRegression dari sklearn dan menggunakan perhitungan manual secara analitik dengan operasi matriks.

Perhitungan analitik dilakukan menggunakan metode Ordinary Least Square (OLS) dengan rumus:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Metode tersebut digunakan untuk mencari garis terbaik yang paling sesuai dengan data yang dimiliki. Hasil dari perhitungan nantinya berupa persamaan regresi yang dapat digunakan untuk melihat hubungan antara variabel X dan Y.

Dataset yang digunakan terdiri dari beberapa pasangan data sederhana. Selanjutnya data akan diproses menggunakan Python untuk mendapatkan nilai koefisien regresi serta membandingkan hasil antara metode sklearn dan metode analitik.

3. Konsep Regresi Linear yang Digunakan#

Pada tugas ini digunakan dua metode untuk menghitung regresi linear, yaitu menggunakan library sklearn dan metode analitik.

3.1 Regresi Linear Menggunakan sklearn#

Metode pertama menggunakan library LinearRegression dari sklearn. Library ini digunakan untuk mempermudah proses perhitungan regresi linear secara otomatis tanpa perlu menghitung matriks secara manual.

Dengan sklearn, program akan langsung mencari nilai:

  • intercept (\(\beta_0\))

  • slope (\(\beta_1\))

berdasarkan data yang diberikan.

Metode ini lebih sederhana dan sering digunakan pada machine learning maupun analisa data karena proses perhitungannya cepat dan praktis.

3.2 Regresi Linear Secara Analitik#

Metode kedua dilakukan secara manual menggunakan operasi matriks. Perhitungan dilakukan menggunakan metode Ordinary Least Square (OLS).

Rumus yang digunakan yaitu:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Keterangan:

  • \(X\) = matriks data

  • \(X^T\) = transpose matriks

  • \((X^TX)^{-1}\) = invers matriks

  • \(Y\) = data target

  • \(\hat{\beta}\) = koefisien regresi

Metode analitik digunakan untuk memahami proses matematika dalam mencari nilai regresi linear secara manual tanpa bantuan perhitungan otomatis dari library.

4. Full Source Code#

import numpy as np
from sklearn.linear_model import LinearRegression

# =========================
# DATA
# =========================

x = np.array([2, 4, 3, 3, 3, 4, 5])
y = np.array([2, 3, 5, 4, 3, 5, 6])

# =========================
# REGRESI LINEAR SKLEARN
# =========================

# Mengubah x menjadi bentuk kolom
x_baru = x.reshape(-1, 1)

# Membuat model regresi
model = LinearRegression()

# Menjalankan regresi
model.fit(x_baru, y)

# Mengambil hasil regresi
beta0 = model.intercept_
beta1 = model.coef_[0]

print("=== HASIL SKLEARN ===")
print("Beta 0 =", beta0)
print("Beta 1 =", beta1)

# Menampilkan persamaan regresi
print("Persamaan Regresi:")
print("Y =", round(beta0, 4), "+", round(beta1, 4), "X")

# =========================
# REGRESI LINEAR ANALITIK
# =========================

# Menambahkan angka 1 di depan data x
X = np.column_stack((np.ones(len(x)), x))

# Membalik baris dan kolom matriks
XT = X.T

# Mengalikan XT dengan X
hasil1 = np.dot(XT, X)

# Membalik matriks
hasil2 = np.linalg.inv(hasil1)

# Mengalikan XT dengan y
hasil3 = np.dot(XT, y)

# Menghitung beta
beta = np.dot(hasil2, hasil3)

print("\n=== HASIL ANALITIK ===")
print("Beta:")
print(beta)

print("Beta 0 =", beta[0])
print("Beta 1 =", beta[1])

# Menampilkan persamaan regresi
print("Persamaan Regresi:")
print("Y =", round(beta[0], 4), "+", round(beta[1], 4), "X")

5. Penjelasan Code#

5.1 Import Library#

import numpy as np
from sklearn.linear_model import LinearRegression

Penjelasan:

  • numpy digunakan untuk membantu operasi array dan matriks.

  • LinearRegression digunakan untuk membuat model regresi linear.

5.2 Membuat Dataset#

x = np.array([2, 4, 3, 3, 3, 4, 5])
y = np.array([2, 3, 5, 4, 3, 5, 6])

Penjelasan:

  • Variabel x berisi data input.

  • Variabel y berisi data output atau target.

5.3 Mengubah Bentuk Data#

x_baru = x.reshape(-1, 1)

Penjelasan:

  • Data x diubah menjadi bentuk kolom karena sklearn membaca data input dalam bentuk matriks.

5.4 Membuat Model Regresi#

model = LinearRegression()

Penjelasan:

  • Digunakan untuk membuat model regresi linear sederhana.

5.5 Menjalankan Model#

model.fit(x_baru, y)

Penjelasan:

  • Model dijalankan menggunakan data X dan Y untuk mencari garis regresi terbaik.

5.6 Mengambil Nilai Regresi#

beta0 = model.intercept_
beta1 = model.coef_[0]

Penjelasan:

  • intercept_ digunakan untuk mengambil nilai konstanta.

  • coef_ digunakan untuk mengambil nilai koefisien regresi.

5.7 Membentuk Matriks X#

X = np.column_stack((np.ones(len(x)), x))

Penjelasan:

  • Menambahkan angka 1 pada kolom pertama matriks.

  • Kolom tersebut digunakan untuk menghitung nilai intercept pada metode analitik.

Hasil matriks:

\[\begin{split} X = \begin{bmatrix} 1 & 2 \\ 1 & 4 \\ 1 & 3 \\ 1 & 3 \\ 1 & 3 \\ 1 & 4 \\ 1 & 5 \end{bmatrix} \end{split}\]

5.8 Membuat Transpose Matriks#

XT = X.T

Penjelasan:

  • Transpose digunakan untuk menukar posisi baris menjadi kolom.

5.9 Menghitung Perkalian Matriks#

hasil1 = np.dot(XT, X)

Penjelasan:

  • Mengalikan matriks transpose X dengan matriks X.

Hasil:

\[\begin{split} X^TX = \begin{bmatrix} 7 & 24 \\ 24 & 88 \end{bmatrix} \end{split}\]

5.10 Menghitung Invers Matriks#

hasil2 = np.linalg.inv(hasil1)

Penjelasan:

  • Digunakan untuk mencari invers dari matriks hasil perkalian sebelumnya.

5.11 Menghitung XTY#

hasil3 = np.dot(XT, y)

Penjelasan:

  • Mengalikan transpose matriks X dengan matriks Y.

Hasil:

\[\begin{split} X^TY = \begin{bmatrix} 28 \\ 103 \end{bmatrix} \end{split}\]

5.12 Menghitung Beta#

beta = np.dot(hasil2, hasil3)

Penjelasan:

  • Digunakan untuk menghitung nilai beta atau koefisien regresi menggunakan metode analitik.

Rumus yang digunakan:

\[ \hat{\beta} = (X^TX)^{-1}X^TY \]

Hasil yang diperoleh:

\[\begin{split} \beta = \begin{bmatrix} 0.4 \\ 1.05 \end{bmatrix} \end{split}\]

6.1 Hasil sklearn#

Beta 1 = 1.05

Y = 0.4 + 1.05 X

6.2 Hasil Analitik#

=== HASIL ANALITIK ===
Beta:
[0.4  1.05]

Beta 0 = 0.3999999999999915
Beta 1 = 1.0500000000000003

Persamaan Regresi:
Y = 0.4 + 1.05 X

7. Persamaan Regresi#

Berdasarkan hasil perhitungan diperoleh persamaan regresi:

\[ Y = 0.4 + 1.05X \]

Artinya:

  • jika nilai X bertambah 1

  • maka nilai Y akan bertambah sekitar 1.05

Karena nilai slope bernilai positif, maka hubungan antara variabel X dan Y juga bersifat positif.

8. Visualisasi Garis Regresi di GeoGebra#

Visualisasi dilakukan menggunakan GeoGebra untuk melihat bentuk garis regresi terhadap data asli.

Titik berwarna biru pada grafik menunjukkan data asli yang digunakan dalam proses regresi linear. Sedangkan garis hijau menunjukkan garis regresi yang diperoleh dari hasil perhitungan.

Persamaan garis regresi yang digunakan yaitu:

\[ y = 0.4 + 1.05x \]

Titik data dimasukkan secara manual ke dalam GeoGebra menggunakan koordinat:

  • A = (2,2)

  • B = (4,3)

  • C = (3,5)

  • D = (3,4)

  • E = (3,3)

  • F = (4,5)

  • G = (5,6)

Setelah itu persamaan garis regresi dimasukkan pada kolom input sehingga garis otomatis muncul pada grafik.

Dari grafik terlihat bahwa garis bergerak naik dari kiri bawah ke kanan atas. Hal tersebut menunjukkan bahwa nilai X dan Y memiliki hubungan positif, dimana kenaikan nilai X cenderung diikuti kenaikan nilai Y.

Selain itu, posisi garis regresi cukup dekat dengan titik data sehingga garis regresi dapat mewakili pola data dengan cukup baik.

9. Kesimpulan#

  1. Regresi linear dapat digunakan untuk melihat hubungan antara variabel X dan Y.

  2. Perhitungan menggunakan sklearn dan metode analitik menghasilkan nilai yang hampir sama.

  3. Nilai intercept yang diperoleh sebesar:

\[ \beta_0 = 0.4 \]

  1. Nilai slope yang diperoleh sebesar:

\[ \beta_1 = 1.05 \]

  1. Persamaan regresi linear yang diperoleh yaitu:

\[ Y = 0.4 + 1.05X \]

  1. Nilai slope yang positif menunjukkan bahwa hubungan antara variabel X dan Y bersifat positif.
