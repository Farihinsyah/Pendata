Data Understanding#

Data Understanding merupakan tahap kedua dalam metode CRISP-DM setelah Business Understanding. Pada tahap ini, fokus utama adalah mengenali dan memahami data yang akan digunakan dalam proyek data mining.

Sebelum melakukan pengolahan atau pembuatan model, praktisi data perlu mengetahui isi, struktur, serta kualitas data yang tersedia. Pemahaman ini penting agar proses analisis selanjutnya dapat berjalan dengan tepat dan sesuai tujuan.

Tahap Data Understanding biasanya terdiri dari beberapa kegiatan utama sebagai berikut:

1. Pengumpulan Data (Collect Initial Data)#

Langkah pertama adalah mengumpulkan data yang relevan dengan kebutuhan proyek. Data dapat berasal dari berbagai sumber, seperti database perusahaan, file laporan, sistem transaksi, atau sumber eksternal lainnya.

Beberapa hal yang diperhatikan pada tahap ini:

  • Sumber data yang digunakan

  • Jumlah data yang tersedia

  • Format penyimpanan data

  • Kesesuaian data dengan tujuan proyek

Pengumpulan data harus dilakukan dengan memastikan bahwa data tersebut memang mendukung analisis yang akan dilakukan.

2. Mendeskripsikan Data (Describe Data)#

Setelah data diperoleh, langkah berikutnya adalah mempelajari karakteristiknya. Pada tahap ini dilakukan pengamatan terhadap struktur dan isi data.

Beberapa hal yang biasanya dianalisis:

  • Jumlah baris dan kolom

  • Nama dan jenis atribut

  • Tipe data (angka, teks, tanggal, dan sebagainya)

  • Gambaran umum isi data

Tujuan dari proses ini adalah untuk memahami bentuk data secara menyeluruh sebelum masuk ke tahap pembersihan.

3. Eksplorasi Data (Explore Data)#

Eksplorasi data dilakukan untuk menemukan pola awal, hubungan antar variabel, atau kemungkinan adanya kejanggalan dalam data.

Kegiatan yang dilakukan antara lain:

  • Melihat distribusi nilai dalam setiap atribut

  • Mengamati hubungan antar variabel

  • Mengidentifikasi pola tertentu

  • Mendeteksi nilai yang tidak wajar

Melalui eksplorasi ini, praktisi data dapat memperoleh gambaran awal mengenai potensi insight yang bisa dihasilkan.

4. Verifikasi Kualitas Data (Verify Data Quality)#

Tahap ini bertujuan untuk mengecek apakah data memiliki masalah yang dapat memengaruhi hasil analisis.

Beberapa permasalahan yang umum ditemukan:

  • Data kosong (missing value)

  • Data ganda (duplikasi)

  • Nilai yang tidak konsisten

  • Kesalahan pencatatan

  • Data yang tidak relevan

Jika ditemukan masalah, maka akan menjadi bahan pertimbangan untuk tahap Data Preparation.

Kesimpulan#

Data Understanding adalah tahap yang berfokus pada pemahaman isi dan kualitas data sebelum dilakukan proses pengolahan lebih lanjut.

Tahap ini membantu memastikan bahwa data yang digunakan benar-benar sesuai dengan kebutuhan proyek. Dengan memahami data secara menyeluruh, proses analisis dan pembuatan model dapat dilakukan dengan lebih efektif dan akurat.

c. Visualisasi Data Menggunakan Scatter Plot#

import seaborn as sns
import matplotlib.pyplot as plt

plt.figure()
sns.scatterplot(
    x='sepal_length',
    y='sepal_width',
    hue='species',
    data=df
)

plt.title("Scatter Plot Sepal Length vs Sepal Width")
plt.show()

Kode tersebut berfungsi untuk menampilkan data dalam bentuk grafik titik agar hubungan antara dua nilai bisa terlihat dengan jelas. Grafik ini membantu memperlihatkan perbandingan antara panjang dan lebar sepal, serta membedakan setiap jenis bunga dengan warna yang berbeda. Dengan tampilan visual seperti ini, pola atau perbedaan antar kelompok data akan lebih mudah diamati dibandingkan hanya melihat angka dalam tabel.

e. Statistik deskriptif#

import pandas as pd
from scipy import stats

df = pd.read_csv("IRIS.csv")

# Hanya kolom numerik
df_numeric = df.select_dtypes(include=['int64','float64'])

for col in df_numeric.columns:
    print("===== Statistik untuk", col, "=====")
    print("Jumlah Data      :", df[col].count())
    print("Rata-rata        :", df[col].mean())
    print("Nilai Minimum    :", df[col].min())
    print("Q1               :", df[col].quantile(0.25))
    print("Median (Q2)      :", df[col].quantile(0.5))
    print("Q3               :", df[col].quantile(0.75))
    print("Nilai Maximum    :", df[col].max())
    print("Kemencengan      :", round(df[col].skew(), 6))

    mode = stats.mode(df[col], keepdims=True)
    print("Modus            :", mode.mode[0], "dengan jumlah", mode.count[0])

    print("Standar Deviasi  :", round(df[col].std(), 6))
    print("Variansi         :", round(df[col].var(), 6))
    print("\n")

6. Eksplorasi Data IRIS Menggunakan Orange#

Alur Analisis Data IRIS Menggunakan Orange#

Pada bagian ini dilakukan proses penelusuran dan pengamatan data dengan memanfaatkan berbagai widget yang tersedia di Orange. Tujuannya adalah untuk mengenali pola, melihat distribusi, serta memahami ciri-ciri dari dataset IRIS sebelum masuk ke tahap analisis yang lebih lanjut.

a. Widget File#

Digunakan untuk memasukkan atau membuka dataset ke dalam Orange. Data bisa berasal dari file seperti CSV, Excel, atau dataset bawaan.

b. Widget Select Column#

Berfungsi untuk memilih kolom mana yang ingin digunakan. Kita bisa menentukan atribut (fitur), target/kelas, maupun meta data sesuai kebutuhan analisis.

c. Widget Rank#

Digunakan untuk memberi peringkat pada fitur berdasarkan tingkat kepentingannya terhadap target. Biasanya dipakai dalam tahap seleksi fitur.

d. Widget Data Table#

Menampilkan data dalam bentuk tabel. Widget ini membantu melihat isi dataset secara langsung, termasuk nilai pada setiap baris dan kolom.

e. Widget Scatter Plot#

Menampilkan grafik sebar (diagram titik) untuk melihat hubungan antara dua variabel. Cocok untuk melihat pola atau pemisahan antar kelas.

f. Widget Correlations#

Menampilkan hubungan atau korelasi antar variabel numerik. Berguna untuk mengetahui apakah ada keterkaitan yang kuat antara satu fitur dengan fitur lainnya.

g. Widget Box Plot#

Menampilkan distribusi data dalam bentuk boxplot. Membantu melihat median, sebaran data, dan kemungkinan adanya outlier.

h. Widget Column Statistics#

Menampilkan ringkasan statistik tiap kolom, seperti rata-rata, minimum, maksimum, dan standar deviasi.

i. Widget Pivot Table#

Digunakan untuk merangkum data berdasarkan kategori tertentu. Widget ini membantu melihat perbandingan atau agregasi data dalam bentuk tabel ringkasan.
