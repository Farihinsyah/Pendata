Analisis Data Obesity Menggunakan Algoritma Decision Tree dan Random Forest#

Pendahuluan#

Decision Tree dan Random Forest merupakan algoritma klasifikasi yang digunakan untuk memprediksi suatu data berdasarkan atribut tertentu. Decision Tree bekerja dengan membentuk pohon keputusan, sedangkan Random Forest merupakan gabungan dari beberapa pohon keputusan untuk meningkatkan akurasi prediksi.

Pada proyek ini dilakukan perbandingan algoritma Decision Tree dan Random Forest menggunakan dataset obesity pada KNIME Analytics Platform.

Dataset yang digunakan berasal dari Kaggle:

https://www.kaggle.com/datasets/ruchikakumbhar/obesity-prediction

Deskripsi Dataset#

Dataset obesity berisi data mengenai karakteristik seseorang yang berhubungan dengan tingkat obesitas.

Kolom target:

Obesity

Kolom tersebut digunakan sebagai label klasifikasi.

3. Python Script Legacy - Training Decision Tree#

Node ini digunakan untuk melakukan preprocessing data dan training model Decision Tree.

Source Code Training Decision Tree#

import pandas as pd
import numpy as np
import os
import pickle

from sklearn.tree import DecisionTreeClassifier
from sklearn.pipeline import make_pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder
from sklearn.impute import SimpleImputer

# Ambil dataset training
dataset_train = input_table_1.copy()

# Nama target
label_target = "Obesity"

# Bersihkan simbol aneh
dataset_train = dataset_train.replace("?", np.nan)

# =========================
# IDENTIFIKASI FITUR
# =========================

# Kolom numerik
numeric_columns = [
    "Age",
    "Height",
    "Weight",
    "FCVC",
    "NCP",
    "CH2O",
    "FAF",
    "TUE"
]

for angka in numeric_columns:
    if angka in dataset_train.columns:
        dataset_train[angka] = pd.to_numeric(
            dataset_train[angka],
            errors="coerce"
        )

# Kolom kategorikal
categorical_columns = [
    "Gender",
    "family_history",
    "FAVC",
    "CAEC",
    "SMOKE",
    "SCC",
    "CALC",
    "MTRANS"
]

for kategori in categorical_columns:
    if kategori in dataset_train.columns:
        dataset_train[kategori] = (
            dataset_train[kategori]
            .astype(str)
            .str.strip()
        )

# Hapus target kosong
dataset_train = dataset_train.dropna(subset=[label_target])

# Pisahkan fitur dan target
fitur_X = dataset_train.drop(columns=[label_target])
target_y = dataset_train[label_target]

# =========================
# PREPROCESSING
# =========================

# Transformasi numerik
numeric_process = make_pipeline(
    SimpleImputer(strategy="median")
)

# Encoder kategorikal
try:
    kategori_encoder = OneHotEncoder(
        handle_unknown="ignore",
        sparse_output=False
    )
except TypeError:
    kategori_encoder = OneHotEncoder(
        handle_unknown="ignore",
        sparse=False
    )

categorical_process = make_pipeline(
    SimpleImputer(strategy="most_frequent"),
    kategori_encoder
)

# Gabungkan preprocessing
gabungan_preprocess = ColumnTransformer([
    ("fitur_angka", numeric_process, numeric_columns),
    ("fitur_kategori", categorical_process, categorical_columns)
])

# =========================
# MODEL DECISION TREE
# =========================

dt_model = DecisionTreeClassifier(
    criterion="entropy",
    max_depth=8,
    min_samples_split=4,
    random_state=10
)

# Pipeline final
pipeline_dt = make_pipeline(
    gabungan_preprocess,
    dt_model
)

# Training model
pipeline_dt.fit(fitur_X, target_y)

# Simpan model
folder_model = os.path.join(
    os.path.expanduser("~"),
    "knime_model"
)

os.makedirs(folder_model, exist_ok=True)

lokasi_model = os.path.join(
    folder_model,
    "dt_obesity_pipeline.pkl"
)

with open(lokasi_model, "wb") as simpan_model:
    pickle.dump(pipeline_dt, simpan_model)

# Output info
output_table_1 = pd.DataFrame({
    "Keterangan": ["Decision Tree selesai dibuat"],
    "Lokasi Model": [lokasi_model],
    "Jumlah Training": [len(dataset_train)],
    "Kolom Target": [label_target]
})

Penjelasan parameter:

  • criterion="entropy" digunakan untuk menghitung kualitas split menggunakan entropy.

  • max_depth=8 membatasi kedalaman pohon agar tidak overfitting.

  • min_samples_split=4 menentukan minimal data untuk melakukan split.

  • random_state=10 digunakan agar hasil training konsisten.

Rumus Entropy#

Entropy(S) = -Σ pi log2(pi)

Keterangan:

  • pi adalah probabilitas kelas ke-i.

  • Semakin kecil nilai entropy maka pemisahan data semakin baik.

6. Python Script Legacy - Training Random Forest#

Source Code Training Random Forest#

import pandas as pd
import numpy as np
import os
import pickle

from sklearn.ensemble import RandomForestClassifier
from sklearn.pipeline import make_pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import OneHotEncoder
from sklearn.impute import SimpleImputer

# Ambil dataset training
dataset_train = input_table_1.copy()

# Nama target
label_target = "Obesity"

# Bersihkan simbol aneh
dataset_train = dataset_train.replace("?", np.nan)

# =========================
# IDENTIFIKASI FITUR
# =========================

# Kolom numerik
numeric_columns = [
    "Age",
    "Height",
    "Weight",
    "FCVC",
    "NCP",
    "CH2O",
    "FAF",
    "TUE"
]

for angka in numeric_columns:
    if angka in dataset_train.columns:
        dataset_train[angka] = pd.to_numeric(
            dataset_train[angka],
            errors="coerce"
        )

# Kolom kategorikal
categorical_columns = [
    "Gender",
    "family_history",
    "FAVC",
    "CAEC",
    "SMOKE",
    "SCC",
    "CALC",
    "MTRANS"
]

for kategori in categorical_columns:
    if kategori in dataset_train.columns:
        dataset_train[kategori] = (
            dataset_train[kategori]
            .astype(str)
            .str.strip()
        )

# Hapus target kosong
dataset_train = dataset_train.dropna(subset=[label_target])

# Pisahkan fitur dan target
fitur_X = dataset_train.drop(columns=[label_target])
target_y = dataset_train[label_target]

# =========================
# PREPROCESSING
# =========================

# Transformasi numerik
numeric_process = make_pipeline(
    SimpleImputer(strategy="median")
)

# Encoder kategorikal
try:
    kategori_encoder = OneHotEncoder(
        handle_unknown="ignore",
        sparse_output=False
    )
except TypeError:
    kategori_encoder = OneHotEncoder(
        handle_unknown="ignore",
        sparse=False
    )

categorical_process = make_pipeline(
    SimpleImputer(strategy="most_frequent"),
    kategori_encoder
)

# Gabungkan preprocessing
gabungan_preprocess = ColumnTransformer([
    ("fitur_angka", numeric_process, numeric_columns),
    ("fitur_kategori", categorical_process, categorical_columns)
])

# =========================
# MODEL RANDOM FOREST
# =========================

rf_model = RandomForestClassifier(
    n_estimators=50,
    criterion="gini",
    max_depth=12,
    min_samples_split=3,
    random_state=15
)

# Pipeline final
pipeline_rf = make_pipeline(
    gabungan_preprocess,
    rf_model
)

# Training model
pipeline_rf.fit(fitur_X, target_y)

# Simpan model
folder_model = os.path.join(
    os.path.expanduser("~"),
    "knime_model"
)

os.makedirs(folder_model, exist_ok=True)

lokasi_model = os.path.join(
    folder_model,
    "rf_obesity_pipeline.pkl"
)

with open(lokasi_model, "wb") as simpan_model:
    pickle.dump(pipeline_rf, simpan_model)

# Output info
output_table_1 = pd.DataFrame({
    "Keterangan": ["Random Forest selesai dibuat"],
    "Lokasi Model": [lokasi_model],
    "Jumlah Training": [len(dataset_train)],
    "Kolom Target": [label_target],
    "Jumlah Tree": [50]
})

Hasil Evaluasi#

Rumus Accuracy#

\[ Accuracy = \frac{Jumlah\ Prediksi\ Benar}{Total\ Data} \]

Keterangan:

  • Accuracy digunakan untuk mengukur tingkat ketepatan model dalam melakukan klasifikasi.

  • Semakin tinggi nilai accuracy maka performa model semakin baik.

Rumus Error Rate#

\[ Error\ Rate = \frac{Jumlah\ Prediksi\ Salah}{Total\ Data} \]

Keterangan:

  • Error Rate digunakan untuk menghitung tingkat kesalahan model.

  • Semakin kecil nilai error rate maka model semakin baik.

Rumus Cohen’s Kappa#

\[ Kappa = \frac{P_o - P_e}{1 - P_e} \]

Keterangan:

  • \(P_o\) = observed agreement

  • \(P_e\) = expected agreement

  • Cohen’s Kappa digunakan untuk mengukur tingkat kesesuaian hasil klasifikasi model.

Rumus Precision#

\[ Precision = \frac{TP}{TP + FP} \]

Keterangan:

  • TP = True Positive

  • FP = False Positive

  • Precision digunakan untuk mengukur seberapa tepat hasil prediksi suatu kelas.

Pada dataset multiclass seperti obesity, precision dihitung untuk setiap kelas kemudian dirata-ratakan.

Rumus Recall#

\[ Recall = \frac{TP}{TP + FN} \]

Keterangan:

  • TP = True Positive

  • FN = False Negative

  • Recall digunakan untuk mengukur kemampuan model dalam menemukan data yang benar pada setiap kelas.

Pada dataset multiclass, recall dihitung untuk seluruh kelas lalu dirata-ratakan.

Rumus F1-Score#

\[ F1Score = 2 \times \frac{Precision \times Recall}{Precision + Recall} \]

Keterangan:

  • F1-Score merupakan gabungan antara precision dan recall.

  • Nilai F1-Score tinggi menunjukkan model memiliki keseimbangan yang baik antara precision dan recall.

F1-Score#

F1-Score merupakan gabungan precision dan recall.

Semakin tinggi nilai F1-Score maka model semakin stabil dalam melakukan klasifikasi seluruh kelas obesity.

Berdasarkan hasil pengujian yang telah dilakukan, algoritma Decision Tree dan Random Forest berhasil digunakan untuk klasifikasi data obesity.

Hasil evaluasi menunjukkan bahwa Random Forest memiliki performa lebih baik dibandingkan Decision Tree. Hal ini dapat dilihat dari nilai accuracy Random Forest sebesar 80.1%, sedangkan Decision Tree sebesar 73.5%.

Selain itu, Random Forest memiliki error rate yang lebih rendah dan nilai Cohen’s Kappa yang lebih tinggi, sehingga model mampu melakukan klasifikasi dengan lebih stabil dan akurat.

Dari sisi precision, recall, dan F1-Score, Random Forest juga menunjukkan performa yang lebih baik karena mampu mengenali berbagai kelas obesity dengan lebih konsisten.

Dengan demikian, model terbaik pada proyek ini adalah Random Forest.
